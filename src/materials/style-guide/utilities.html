There are some general-use utility classes available for use on content types and layouts.

Image Dimming

The .has-dim class is used to place a CSS opacity layer above an image, and was originally developed for use with Jumbotrons and Heroes.

Dimming Levels

Classes are incremented by their opacity level:

  • .has-dim (.5 opacity)
  • .has-dim-2 (.2 opacity - base style)
  • .has-dim-4 (.4 opacity)
  • .has-dim-6 (.6 opacity)
  • .has-dim-8 (.8 opacity)

Legacy Headers

The .legacy-headers class is used on the #content element to reinstate the opinionated headers in the pre-2.0 design.
