Overview

This UI toolkit is a highly-modular design system for rapid web page development. It contains different materials that can be assembled into more complex page layouts.

The SCU website calls its materials content types, some of which operate by themselves, and some of which can be assembled to create something whole on a larger scale (for example, a Story Display of several Story types).

By using content types, content creators are able to piece together webpages by focusing solely on the content that pertains to their part of the website. The rest of the content forms the headers, footers, and navigation, which are described as pages that are included outside of the scope of content types.

This guide contains real working examples, code snippets, documentation, and style guidelines.
